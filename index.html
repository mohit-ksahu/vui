Introduction

A collection of ready-to-use UI components built with plain HTML, CSS, and JavaScript.

Features

  • No external dependencies — entirely self-contained
  • Framework-agnostic — works with React, Vue, or plain HTML
  • Design-token driven — easy customization via CSS variables

Installation

Include the library in your project by copying the components or referencing compiled bundles.

1. Add files

<link rel="stylesheet" href="path/to/components/index.css">
<script type="module" src="path/to/components/index.js"></script>

2. Minified Bundles (optional)

For production, you can compile and reference minified bundles (index.min.js and index.min.css) using esbuild:

npx esbuild components/index.js --bundle --minify --format=esm --outfile=index.min.js
npx esbuild components/index.css --bundle --minify --outfile=index.min.css

Then include the minified outputs in your HTML:

<link rel="stylesheet" href="path/to/index.min.css">
<script type="module" src="path/to/index.min.js"></script>

Theme Management & Utilities

Toggle elements dynamically using utility classes, or control theme state programmatically.

<span class="dark:hidden">Visible in light mode</span>
<span class="hidden dark:block">Visible in dark mode</span>
import { setTheme, switchTheme } from './index.js';

// Toggle or force themes
switchTheme(); 
setTheme('dark'); // 'light', 'dark', or 'system'

// Listen for updates
document.addEventListener('theme:change', (e) => {
  console.log(e.detail.isDark); // true/false
  console.log(e.detail.theme);  // 'light' | 'dark' | 'system'
});
